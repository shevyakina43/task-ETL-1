# Рекомендовані методи Pandas (що використовувати)

- pd.read_csv, pd.read_excel — для читання даних

- df.head(), df.info(), df.describe() — первинне дослідження

- df.isna(), df.duplicated() — перевірка пропусків / дублікатів

- df.drop() — видалення колонок / рядків

- df.set_index(), df.reset_index() — маніпуляція індексами

- df.insert() — для вставки нових колонок

- Прямі арифметичні операції (наприклад, df["A"] + df["B"]) — для обчислення нових колонок

- pd.concat або df.append() — для додавання нових рядків

- df.loc[] — для фільтрації за мітками або логікою

- df.iloc[] — для позиційної вибірки

- df.groupby().agg() — для групування та агрегування (при потребі)

- df.to_csv(), df.to_excel() — експорт даних

- df.apply(lambda ...) — якщо логіка обчислення дуже специфічна
